# HealthCompass-AI

An AI-powered healthcare assistant built using Flask, Machine Learning, and Natural Language Processing that helps users analyze symptoms, predict possible diseases, estimate urgency levels, and connect with relevant medical specialists.

⸻

🚀 Project Overview

HealthCompass AI is a smart healthcare web application designed to provide preliminary symptom analysis using Machine Learning.

The system allows users to:

* Describe symptoms in natural language
* Receive AI-generated disease predictions
* View confidence scores for top possible conditions
* Get urgency/severity estimation
* Receive medical advice suggestions
* Find nearby specialists
* Maintain analysis history through a dashboard
* Download AI-generated PDF health reports

The project combines:

* Machine Learning
* NLP (Natural Language Processing)
* Flask Backend Development
* Database Management
* Modern Responsive Frontend Design

⸻

✨ Key Features

🤖 AI Symptom Analysis

Users can enter symptoms in natural language such as:

fever, cough, headache, body pain

The AI model predicts possible diseases using TF-IDF vectorization and a Random Forest classifier.

⸻

📊 Top Disease Predictions

Instead of returning only one disease, the system shows:

* Top 3 possible diseases
* Confidence percentages
* Probability visualization bars

Example:

Influenza — 42% 
COVID-19 — 31% 
Common Cold — 27%

⸻

🚨 Urgency Detection

The application estimates urgency level based on model confidence and disease severity.

Urgency levels include:

* Low
* Moderate
* High
* Emergency

⸻

🧑‍⚕️ Doctor Recommendation System

Based on predicted conditions, the system recommends suitable medical specialists.

Examples:

Disease	Recommended Specialist
Migraine	Neurologist
Diabetes	Endocrinologist
Asthma	Pulmonologist

The app also provides a Google Maps integration to help users locate doctors nearby.

⸻

🧠 Machine Learning Pipeline

The AI model is trained using:

Component	Technology
Vectorization	TF-IDF Vectorizer
Classifier	Random Forest
NLP	Natural Language Symptom Processing
Model Persistence	Joblib

Model Workflow

1. User enters symptoms
2. Symptoms are preprocessed
3. TF-IDF converts text into vectors
4. Random Forest predicts diseases
5. Probabilities are generated
6. Top predictions are displayed

⚙️ Installation Guide

1️⃣ Clone Repository

git clone https://github.com/Tushark6038/healthcompass-ai.git

⸻

2️⃣ Move into Project Folder

cd healthcompass-ai

⸻

3️⃣ Install Dependencies

pip install -r requirements.txt

⸻

4️⃣ Train the AI Model

python train_model.py

This generates:

health_model.pkl

⸻

5️⃣ Run Flask Server

python app.py

⸻

6️⃣ Open in Browser

http://127.0.0.1:5001
